# Digital Compost Simulator

## Description
An iOS app that simulates a digital compost bin, allowing users to input ingredients and predict the types of bacterial, fungal, and viral life present.

## Features
- Input ingredients and track microbial life.
- Reset the simulation with a "Dump Bucket" button.
- Educational insights about microbes.

## Technologies Used
- SwiftUI
- Core ML (for microbial predictions)
- Combine (for reactive updates)
